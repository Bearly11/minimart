# mini mart (Mart Management System)

## Features
- User authentication (JWT + refresh token)
- Token rotation & revocation
- Monolithic architecture

## Tech Stack
- Flask
- Flask-JWT-Extended
- SQLAlchemy
- SQLite

## How to run
- pip install -r requirements.txt
- flask run (starts the application)
